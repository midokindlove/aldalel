# 🎮 MTA San Andreas - دليل الأوامر

موقع شامل لأوامر وقوانين سيرفر MTA San Andreas

## ✨ المميزات

- 🤖 بوت ذكي للرد على الاستفسارات
- 📋 عرض جميع الأوامر مع إمكانية البحث والفلترة
- 📜 عرض القوانين باللغتين الإنجليزية والعربية
- 🏠 معرض صور الانترو مع إمكانية نسخ الأسماء
- ⚙️ لوحة تحكم للإدارة (الرقم السري: 0000)
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 📋 زر نسخ للأوامر والقوانين

## 🚀 طريقة النشر على Netlify

1. ارفع الملفات إلى GitHub
2. سجل الدخول إلى Netlify
3. اختر "New site from Git"
4. اختر المستودع
5. اضبط الإعدادات:
   - Build command: (فارغ)
   - Publish directory: (الجذر)
6. اضغط Deploy
